The wakoopa widget allows you to easily show your readers what software you use via the [wakoopa](http://wakoopa.com/ "wakoopa - Software gone social") service.

== Description ==

[wakoopa](http://wakoopa.com/ "wakoopa - Software gone social") is a social website centered around sharing what software you use.  This widget displays your top 10 used pieces of software in either image or plain text form.

== Installation ==

1. Upload `wakoopa.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the widget to your theme via the Presentation -> Sidebar widgets screen
1. Be sure to input your wakoopa username into the settings for the widget
